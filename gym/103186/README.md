## B

我想用最小费用最大流，但是复杂度过不了

先考虑只有两个的情况，那么按照 $b_i - a_i$ 的值排序，然后前 a 个选为第一类，后 b 个为第二类

现在有三个，我们依然按照前两个排序，那我们就 $f(i, j)$ 表示前 i 个选 $j$ 的答案值，然后剩下的 $i - j$ 个就按照贪心解决（所以是先给 $a$ 没毛病)

> 官方题解的 $O(n \log n)$ 的做法暂时未知
