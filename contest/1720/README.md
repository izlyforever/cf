$a_{b_p} \oplus b_{p+1} < a_{b_{p+1}} \oplus b_p$ 等价于高位一致，之后一位 $a_{b_{p+1}} \oplus b_p = 1$ 而 $a_{b_p} \oplus b_{p+1} = 0$。而高位一致等价于对于较大的 $i$，$(a_{b_{p+1}} \oplus b_{p+1}) \And 2^i = (a_{b_p} \oplus b_p) \And 2^i$。而不同的那一位，需要根 $b_{p + 1}$ 相同。所以可以用  Trie 完美解决
