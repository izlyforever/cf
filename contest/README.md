- $10^6$ 最多可支持 $O(n \log n)$，
- $3 \cdots 10^5$ 最多可支持 $O(n \sqrt{n})$, $O(n \log^2 n)$
- $10^5$ 最多可支持 $O(\frac{n^2}{64})$(即 bitset 或者向量集优化)
- $10^4$ 最多可支持 $O(n^2 \log n)$
- $10^3$ 最多可支持 $O(n^{\frac{5}{2}})$，或常数极小的 $O(n^3)$ 例如 floyd, 图论闭包，矩阵乘法
