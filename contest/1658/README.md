## A

## B

假设 $p | gcd$，那么最多有 $\lfloor \frac{n}{p} \rfloor$ 个。因此 $p = 2$ 是唯一的可能，且此时 $n$ 为偶数，然后答案为 $(\frac{n}{2}!)^2$
