## B

- 如果 $n$ 为奇数，那么显然只要先手第一次随便找个位置取完必赢
- 否则，考虑奇偶的位置各自最小值如果不一样，先手选择其中大的必胜
- 否则，既然奇偶各自最小值一样，那么随便找一个最小值，一直往左边找，直到找到比最小值大的，先手选择这个位置必胜（除非找不到）
- 即后手只有全相等，且 $n$ 为偶数才能赢
