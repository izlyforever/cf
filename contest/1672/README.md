## D

枚举 `b`，如果当前 `a` 对应位置和 `b` 相等，那么各自下标加 1，否则

$a, b$ 下标分别为 $i, j$。
- 若 $a[i] = b[j]$，各自下标自增
- 否则若 $b[j] = b[j - 1]$，且 map 中有这个值，就把他丢出来，$j$ 下标自增
- 否则下标 i 递增直到 $a[i] = b[j]$


## E
