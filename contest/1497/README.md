### E：E1 贪心即可，然后 E2 又是个 DP

我们考虑 `Dp[i][j]` 为删除 j 个节点后，前 i 个最少可以分成多少个。那么显然 $DP[i][j] = \min DP[i_x][j - x]$，其中 $i_x, \cdots, i$，$i_x$ 删除 x 个 点后， $i_x, \cdots, i$ 所有数互异的最小的值。然后我们可以先求 $i_x$，求的时候注意细节，否则 TLE
